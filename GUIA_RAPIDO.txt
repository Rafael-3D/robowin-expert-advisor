⚙️ PASSO 2: CONFIGURAÇÃO INICIAL
─────────────────────────────────

Configure os níveis de preço (ordem crescente, múltiplos de 5):

┌─────────────────────────────────────┐
│  pontoVenda2:  190300  ← Venda mais │
│  pontoVenda1:  190100  ← agressiva  │
│                                     │
│  pontoCompra1: 189700               │
│  pontoCompra2: 189500  ← Compra mais│
└─────────────────────────────────────┘

Gestão de risco:
- takeProfit: 600 pontos
- stopLoss: 200 pontos
- breakEvenPontos: 350 pontos

Horário:
- horaInicio: "09:10"
- horaFim: "17:30"

Comportamento de ordens:
- usarOrdemMercado: false (recomendado)
  └─ false = só entra no preço exato (conservador)
  └─ true  = entra a mercado se nível passou (agressivo)

📊 PARÂMETRO usarOrdemMercado
──────────────────────────────

Controla o que fazer quando o preço JÁ PASSOU do nível:

┌─────────────────────────────────────────────────────────┐
│  Exemplo: Nível compra = 187620, Preço atual = 187300   │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  usarOrdemMercado = false (🛡️ CONSERVADOR)              │
│  └─ NÃO envia ordem, aguarda preço voltar               │
│                                                         │
│  usarOrdemMercado = true (⚡ AGRESSIVO)                 │
│  └─ COMPRA A MERCADO em 187300                          │
│  └─ ⚠️ Risco de slippage!                               │
│                                                         │
└─────────────────────────────────────────────────────────┘

💡 Recomendação: USE false (padrão) para maior segurança
